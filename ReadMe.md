# SPW Lysimeter Analysis

This repository contains the R scripts and documentation for the analysis of lysimeter data associated with the publication:

"Impacts of groundwater level management on geochemical processes at an agricultural site with shallow groundwater"
Jörg Steidl*, https://orcid.org/0000-0002-6599-0450; Ottfried Dietrich, https://orcid.org/0000-0003-4637-9784; Christoph Merz, https://orcid.org/0000-0001-7434-9828
a Leibniz Centre for Agricultural Landscape Research (ZALF), Eberswalder Straße 84, 15374 Müncheberg, Germany
b Hydrogeology Group, Institute of Geological Sciences, Freie Universität Berlin, Malteserstr. 74–100, 12249 Berlin, Germany

Journal: Environmental Earth Sciences

*Corresponding author.
E-mail address: jsteidl@zalf.de
Current address: Leibniz Centre for Agricultural Landscape Research (ZALF), Eberswalder Straße 84, 15374 Müncheberg, Germany.

## Project Structure
- R scripts for data processing and statistical analysis.
- Source data for the lysimeter measurements are available at `https://doi.org/10.4228/zalf-vevz-ys85`.
- `renv.lock`: File for restoring the exact R package environment.

## Prerequisites
To reproduce the analyses, `renv` must be installed.
The R environment is automatically set up using the package versions defined in the `renv.lock` file.
